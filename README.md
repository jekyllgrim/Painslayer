*Painslayer* is a gameplay mod for GZDoom engine by Agent_Ash aka Jekyll Grim Payne, inspired by *Painkiller* (2004) by People Can Fly. 

*Painslayer* requires [GZDoom 4.6.0](https://zdoom.org/downloads).
